Detailed Description

A functor used to calculate the boids new position using the GPU.

Called using a thrust functor and executed on the GPU for each boid. Updates the boids poisition and velocity by applying the boids current acceleration. The boid is also allocated a new cell index based off its new location. If the boid goes out of bounds, this functor moves it to the opposite side of the world.

__host__ __device__ thrust::tuple<int, float4, float4> calculate_position::operator() ( thrust::tuple< float4, float4, float4 >  input)
inline

Updates each boid on the GPU, calculating a new position, velocity and cell index.

Executed ON THE GPU within a thrust transform call for EACH BOID. Acceleration is applied to velocity and velocity is applied to position, taking into account elapsed time. If max velocity is reached, the velocity is capped. If the boid leaves the world space, it is place back on the opposite side of the world. Tuple input and ouput of functor ensures improved memory accesses due to memory coalescing.

Parameters
inputA tuple containing the position, velocity and acceleration of the boid
Returns
tuple<int, float4, float4> Containing the new cell index, position and velocity
