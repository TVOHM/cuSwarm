void simulation::calculatePositions ( int  elapsed)
private

Updates every boids position and velocity on the GPU using the time elapsed, also allocates every boid a cell index based upon its position.

Remarks
If an agent exceeds the world bounds it is spawned on the other side of the world
If an agent exceeds max_velocity, its velocity is capped
