Detailed Description

Static camera object used to control viewing.

Stores camera position and looking vectors as well as the key states of the simulation. Handles all user key presses via the keyDown and keyUp functions. Takes appropriate action in update() depending on the state of keys (the keys that are pressed). This is a static class and should not be instanciated.

void camera::update ( int  elapsed)
static

If the user is pressing the W or S keys, the camera is moved backwards and forwards respectivly at a rate of camera_velocity per second If the user is pressing the A or D keys, the camera is strafed left and right respectivly at a rate of camera_velocity per second If the user is pressing the 4 or 6 keys, the camera is rotated left and right respectivly on the x axis at a rate of camera_rotation degrees per second If the user is pressing the 8 or 5 keys, the camera is rotated up and down on the y axis at a rate of camera_rotation degrees per second If the user is pressing the R key, the current config will be reloaded from 'config.xml'. This feature is used to change simulation parameters during the simulation. Adjusts simulation camera parameters based upon user input

Parameters
elapsedThe time since the last update in miliseconds
